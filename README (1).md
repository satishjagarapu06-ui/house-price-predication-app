 # House Price Prediction using Ames Housing Dataset

This project develops a machine learning model to predict house prices using the Ames Housing dataset.

## Files
- `app.py` - Streamlit web app
- `best_ames_house_price_model.pkl` - trained model
- `AmesHousing.csv` - dataset
- `requirements.txt` - required libraries
- `houseprice.ipynb` - project notebook

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
